# projetoFinch
--------------------------Instruções--------------------------

# Nescessário instalar Python 3.7, Virtualenv

#1 Com os dois acima instalados em um terminal Linux, execuar os comandos:
    -$ cd projetoFinch
    -$ source venv/bin/activate
    -$ python finch.py

#2  Abrir um browser 

#3 Na Url digitar:
    -localhost:5000
